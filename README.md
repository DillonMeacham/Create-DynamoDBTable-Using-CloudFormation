<h1>Create a DynamoDB Table Using CloudFormation</h1>

<h2>Description</h2>
In this project, Alfredo of Alfredo's Pizza is building a web application to manage his supply and food inventory. He's looking for help to provision his database infrastructure, which will hold his data. To assist him, I'll use CloudFormation to write the desired state of the infrastructure using code, which includes a DynamoDB table to hold inventory items. <br />


<h2>Environments Used </h2>

- <b>AWS CloudFormation</b>
- <b>AWS DynamoDB</b>
